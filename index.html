HTML Table Reference

Semantic HTML

Tag Name Description
<header> header container either for navigational links or introductory content
<nav> navigation used to define a block of navigation links
<main> main used to encapsulate the dominant content within a webpage
<footer> footer content at the bottom of the subject information
<section> section defines elements in a doc such as chapters headings etc.
<article> article holds contents that makes sense on its own
<aside> aside mark additional info that can enhance another element but isn't required to understand the main content
